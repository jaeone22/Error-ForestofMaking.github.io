광고로부터 탈출

저도 광고가 나오는게 얼마나 싫은지 잘 알고있어요
그래서 goUP에는 강제 광고가 단 한개도 포함되어있지 않아요
앱 사용중 광고때문에 불편한 경험은 없을거에요

데이터는 안전하게

서버로 전송되는 데이터들은 모두 암호화되어 안전하게 보관되어요
비밀번호는 SHA-256으로 복호화가 불가능하게 암호화하고, 다른 정보들은 AES로 안전하게 보관해요
그래서 서버에 있는 데이터는 아무도 알수 없어요

개인정보의 자유

goUP 앱, 프로그램, 사이트에선 개인정보를 수집하지 않아요
goUP ID 게정을 만들거나, 동기화할때는 어쩔수 없이 개인정보가 필요하기 때문애 사용해요

피드백은 빠르게

2일 안에 모든 피드백이나 문의사항을 답변하고, 일주일 안에 피드백 사항을 적용하고있어요
문의는 Discord 서버, Telegram 서버, 삭제 피드백 설문조사에서 할수 있어요

사용은 자유롭게

다른 앱이나 프로그램을 설치하거나 사용하라고 강요하지 않아요
goUP ID 게정을 만들라고 강요하지도 않아요
그냥 사용하고 싶은 만큼, 원하는대로 사용할수 있어요
