다운로드 폴더의 'Setup3.exe' 파일을 실행해 설치해주세요

⚠️ Windows의 PC 보호 창 해결방법
일부 컴퓨터에서 "Windows의 PC 보호" 창이 뜨며 설치가 진행되지 않는 경우 "자세히 보기"를 클릭 후 "실행"을 눌러주세요

⚠️ Windows 디펜더 예외처리
Windows 디펜더에서 goUP Setup3.exe를 악성코드로 잘못 탐지하는 일이 일어나는 경우 해당 파일을 예외로 설정하고 복구해주세요
Windows 디펜더가 파일을 삭제한경우 추후 goUP Store를 삭제할때 수동으로 https://goup.im 에서 설치관리자를 다운로드해야해요
